# Go API REST Basic

### Routes:

- GET : [/v1/api/books](/v1/api/books)
  - Returns all the books
- POST : [/v1/api/books](/v1/api/books)
  - Create a new book.
- PUT : [/v1/api/books/{id}](/v1/api/books/{id})
  - Update a book.
- DELETE : [/v1/api/books/{id}](/v1/api/books/{id})
  - Delete a book.
